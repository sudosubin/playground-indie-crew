# Korean Universities and Scholarship Notice Boards

> A comprehensive list of colleges and universities in Korea with their academic and scholarship notice board URLs.

## Overview

This document contains a curated list of Korean universities along with verified URLs for their academic notices (학사공지) and scholarship notices (장학공지).

### Verification Status Legend

- ✅ **Verified** - URL accessed and confirmed to display meaningful information
- ⚠️ **Partial** - URL accessed but requires authentication or has limited access
- ❌ **Invalid** - URL returned error or did not load
- ⏳ **Pending** - URL not yet verified

**Website**: https://academyinfo.go.kr

This is the **official government portal** operated by the Korean Council for University Education (KCUE, 한국대학교육협의회) under the Ministry of Education.

**Features**:

- Complete list of all 459 universities (250 4-year + 165 2/3-year + 44 other institutions)
- Regional breakdown available

## Data Collection Methodology

### Official Data Sources

1. **Academy Information Portal (대학알리미)** - https://academyinfo.go.kr
   - Most authoritative source
   - Provides complete list with Excel download
   - Updated regularly by the government

2. **Korean Council for University Education (KCUE)** - https://www.kcue.or.kr
   - Industry association for universities
   - Links to the Academy Information portal

3. **Korean Ministry of Education** - https://english.moe.go.kr/
   - Government department responsible for education policy

### URL Patterns

Common URL patterns for Korean university notice boards:

1. **Academic Notices (학사공지)**:
   - `https://[university].ac.kr/notice` or `/academic`
   - `https://[university].ac.kr/board/notice`
   - `https://[university].ac.kr/community/notice`

2. **Scholarship Notices (장학공지)**:
   - `https://[university].ac.kr/scholarship`
   - `https://[university].ac.kr/academics/resources/scholarships`
   - `https://[university].ac.kr/financial-aid`

### Categories of Universities

1. **4-Year Universities (대학교)**: 250 institutions
2. **2/3-Year Colleges (전문대학)**: 165 institutions
3. **Other Institutions (기타)**: 44 institutions (education universities, cyber universities, etc.)

## How to Get Complete University List

### Method 1: Official Excel Download

1. Visit https://academyinfo.go.kr/intro/intro0350/intro.do
2. Click "공시대상학교목록 다운로드" (Download Disclosure Target School List)
3. Or click "엑셀다운로드" (Excel Download) button

### Method 2: API Access

The Academy Information portal provides data through the Korean Open Data Portal:

**Dataset**: "대학정보 공시"

- URL: https://www.data.go.kr/data/15107738/openapi.do
- Provides university information in JSON/XML format
- Requires API key registration

**Available Data**:

- University basic information (school name, address, type)
- Admission information
- Tuition and scholarship data
- Employment statistics
- Faculty and student ratios

### Method 3: Direct Excel Download

The Academy Information portal provides a direct Excel download:

```bash
# Download URL (requires session cookie)
https://academyinfo.go.kr/intro/intro0350/downloadExcel.do
```

### Method 4: Web Scraping

Use the regional filter buttons on the portal to view universities by region, then extract the displayed lists.

**Example using curl**:

```bash
# Get university list by region
curl -s "https://academyinfo.go.kr/intro/intro0350/selectList.do?regionCode=11" \
  -H "Accept: application/json" \
  -H "Content-Type: application/json"
```
